# BackLens Trademark and Branding

This document describes permitted and prohibited uses of the BackLens name, logo, and branding.

## Scope

The project source code is licensed under AGPL-3.0-only.
Trademark and branding rights are separate from the code license.

## Allowed Uses

You may use the BackLens name to:

- refer factually to this upstream project
- describe compatibility, such as "works with BackLens"
- link to this repository in documentation and tutorials

## Not Allowed Without Permission

You may not, without prior written permission:

- use BackLens logos or branding in a way that implies endorsement
- name a fork or commercial offering in a confusingly similar way
- present a modified version as the official BackLens project
- use BackLens marks in deceptive or misleading marketing

## Forks and Derivatives

Forks are allowed under AGPL, but should use distinct naming and branding when distributed publicly, especially in commercial contexts.

## Reservation of Rights

All trademark and branding rights in the BackLens name and logos are reserved by their respective owners.
