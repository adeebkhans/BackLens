# BackLens Examples

This folder contains system-level sample repositories used to validate BackLens end-to-end.

## Repositories

### express-backend-demo
Primary showcase repository for BackLens. Use this for docs, screenshots, demos, and regression checks across parser, graph-store, API, web, and VS Code extension flows.

### layered-ts-demo
Secondary TypeScript architecture sample focused on layered design and dependency flow across controller, service, repository, and contract boundaries.

## Main Demo Repo

Use `express-backend-demo` as the primary docs and screenshot repository to keep examples consistent.

## Scope Rule

- Parser-only edge-case assets belong in packages/parser/fixtures.
- End-to-end product validation assets belong in examples/.
